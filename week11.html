第11周：团队创意项目

教案

教学目标

  • 激发学生创意，设计独特机器人任务
  • 培养团队协作与项目管理能力
  • 完成并展示创意项目成果

教学时间

总时长：2小时

  • 理论讲解：30分钟
  • 实践操作：80分钟
  • 总结与答疑：10分钟

教学内容

  1. 理论部分（30分钟）
    • 项目概述（10分钟）
    • 团队分工与计划（10分钟）
    • 实现方法（10分钟）
  2. 实践部分（80分钟）
    • 任务1：项目头脑风暴（20分钟）
    • 任务2：程序设计与实现（40分钟）
    • 任务3：测试与准备展示（20分钟）
  3. 总结与答疑（10分钟）

教学方法

PPT展示+创意项目案例+分组合作

评估方式

  • 检查任务1中项目主题是否明确
  • 验证任务2中代码与硬件是否正常运行
  • 观察任务3中展示准备情况
  • 提问：团队协作中遇到的问题如何解决？

所需材料

  • 电脑（每组1台）
  • 机器人硬件（传感器、执行器、摄像头等）
  • PPT课件
  • Python与LeRobot环境
  • 辅助材料（积木、彩纸等）

教材内容

第11章：团队创意项目

11.1 项目概述

目标：设计一个独特的机器人任务。

示例：

  • 机器人舞蹈：按节奏移动
  • 物体分拣：按颜色分类

11.2 团队分工与计划

角色：

  • 程序员：编写控制代码
  • 硬件师：连接传感器与执行器
  • 测试员：调试与验证

时间表：

  1. 头脑风暴（20分钟）
  2. 实现代码与硬件（40分钟）
  3. 测试与优化（20分钟）

11.3 实现方法

综合利用传感器、执行器和视觉。

示例（机器人舞蹈）：

import lerobot
robot = lerobot.connect("robot")
for i in range(3):
    robot.move_forward(1)
    robot.turn_left(90)
    time.sleep(0.5)

11.4 实践任务

任务1：头脑风暴

团队讨论并记录项目主题（如“颜色分拣”）。

任务2：程序设计与实现

示例（颜色分拣）：

import lerobot
robot = lerobot.connect("arm")
while True:
    pos = robot.detect_object("red")
    if pos:
        robot.move_to(pos[0], pos[1], pos[2])
        robot.grasp()
        robot.move_to(0, 0, 0.1)  # 放下位置
        robot.release()

任务3：测试与准备展示

运行程序并录制视频。

11.5 小结

创意项目展示综合能力，团队协作是成功关键。

课后练习

  1. 写下团队分工和时间计划。
  2. 优化项目代码，添加新功能。
