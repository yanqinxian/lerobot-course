教材内容

第3章：传感器与数据采集

3.1 什么是传感器？

传感器是机器人感知环境的“眼睛”和“耳朵”，将物理量转换为电信号。

3.2 常见传感器类型

  • 距离传感器：测量物体距离（如超声波传感器）
  • 颜色传感器：识别物体颜色
  • 摄像头：捕捉图像或视频

3.3 传感器在机器人中的作用

提供数据给控制器，用于决策，如避障、导航。

3.4 LeRobot中传感器操作

使用LeRobot读取数据：

  • 距离传感器：robot.get_distance()
  • 颜色传感器：robot.get_color()
import lerobot
robot = lerobot.connect("robot")
distance = robot.get_distance()
print("距离:", distance)

3.5 实践任务

任务1：读取距离传感器数据

import lerobot
import time
robot = lerobot.connect("robot")
for i in range(5):
    distance = robot.get_distance()
    print("障碍物距离:", distance, "厘米")
    time.sleep(1)

任务2：读取颜色传感器数据

import lerobot
robot = lerobot.connect("robot")
color = robot.get_color()
print("检测到的颜色:", color)

任务3：综合应用

import lerobot
import time
robot = lerobot.connect("robot")
while True:
    distance = robot.get_distance()
    if distance < 10:
        print("警告：障碍物太近！")
    else:
        print("安全距离:", distance)
    time.sleep(1)

3.6 小结

传感器是机器人感知世界的关键，通过LeRobot可轻松获取数据。

课后练习

  2. 修改任务3代码，当距离小于5厘米时停止循环。
