教学大纲

第1周：认识机器人与LeRobot

目标

了解机器人基本概念及LeRobot框架简介

内容

  • 什么是机器人？（传感器、执行器、控制器）
  • LeRobot的特性与优势
  • 安装与配置LeRobot环境

实践

运行LeRobot示例程序（如机械臂简单移动）

第2周：Python编程基础

目标

掌握Python基础，为LeRobot编程打基础

内容

  • 变量、循环、条件语句
  • 函数与模块

实践

编写简单Python程序控制机器人LED灯闪烁

第3周：传感器与数据采集

目标

理解传感器工作原理及数据处理

内容

  • 传感器类型（距离、颜色、摄像头等）
  • LeRobot中读取传感器数据的方法

实践

用传感器检测障碍物并输出数据

第4周：执行器与运动控制

目标

学会控制机器人运动

内容

  • 电机与伺服机的原理
  • LeRobot中运动控制API

实践

编程让机器人完成直线移动和转弯

第5周：条件与循环逻辑

目标

掌握程序控制逻辑

内容

  • if-else语句与while循环
  • 逻辑在机器人控制中的应用

实践

编写程序让机器人避开障碍物

第6周：基础项目 - 迷宫导航

目标

综合运用所学知识完成小项目

内容

  • 项目需求分析与设计
  • 传感器与运动控制结合

实践

让机器人自主走出简单迷宫

第7周：摄像头与视觉基础

目标

引入视觉处理基础

内容

  • 摄像头数据获取
  • 简单图像处理（如颜色检测）

实践

编程让机器人跟随特定颜色移动

第8周：模仿学习简介

目标

理解LeRobot的核心功能——模仿学习

内容

  • 什么是模仿学习？
  • LeRobot中模仿学习的基本流程

实践

录制并重现机械臂的简单动作

第9周：进阶项目 - 物体抓取

目标

完成视觉与运动结合的复杂任务

内容

  • 视觉识别与坐标计算
  • 机械臂抓取逻辑

实践

让机械臂识别并抓取指定物体

第10周：优化与调试

目标

提升程序健壮性与效率

内容

  • 调试技巧
  • 代码优化方法

实践

优化前几周的项目代码

第11周：团队创意项目

目标

培养团队协作与创新能力

内容

  • 分组设计并实现创意机器人任务
  • 项目展示准备

实践

完成团队项目（如机器人舞蹈或分拣任务）

第12周：成果展示与总结

目标

总结学习成果并展望未来

内容

  • 各小组展示项目
  • 机器人编程的未来发展

实践

评选最佳项目并颁发证书
